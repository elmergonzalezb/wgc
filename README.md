# wgc
Create clients and Wireguard server configuration

## Download
```
mkdir -p /etc/wireguard ; cd /etc/wireguard
```

```
wget https://raw.githubusercontent.com/uGeek/wgc/master/wgc
```

```
sudo chmod +x wgc
```

```
./wgc
```
